# My Tools

AI 驱动的实用工具集合。每个工具独立一个文件夹，可独立运行。

## 工具列表

| 工具 | 说明 | 技术栈 |
|------|------|--------|
| [lexiangla-scraper](./lexiangla-scraper/) | 抓取腾讯乐享文档/PPT/Word，导出 HTML、Markdown、原始文件或 8K 高清预览图 + PDF | Python + Playwright + Pillow |

## 使用方式

进入对应工具文件夹，参照其 `README.md` 进行安装和使用。

## 新增工具

1. 在根目录创建新文件夹（kebab-case 命名）
2. 实现功能并编写 `README.md`
3. 更新本文件的工具列表
